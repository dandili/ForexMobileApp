User Guide

Landing page/Login

This is where you enter your username and password (eee, eee). We can see it has the logo of the app.


Dashboard

This is the dashboard. You select which section of the app to proceed to. To return or to log out you must press back. Because it was not a full requirement to have a login using a database it is therefore hard coded for now.


News

This is the news section of the app. It automatically refreshes the feed when you go onto the News section. It is powered by the newsapi.org. It uses fxstreet.com as a source for the news information. This is a trusted FOREX website.


Calendar

The calendar section allows you to select currency pairs and allocate them to a day. This is made possible through SQLite. There is also a reminder button in the bottom right which will push a notification to the user in an hour.


Learn

The learn section is leads the user outside of the app into BabyPips.com. A well trusted FOREX education website.
